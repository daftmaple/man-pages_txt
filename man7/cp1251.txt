NAME
       cp1251 - CP 1251 character set encoded in octal, decimal, and hexadeci-
       mal

DESCRIPTION
       The Windows Code Pages include several 8-bit extensions	to  the  ASCII
       character set (also known as ISO 646-IRV).  CP 1251 encodes the charac-
       ters used in Cyrillic scripts.
